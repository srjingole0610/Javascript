Temperature Forecast Challenge

Create a temperature forecast display system!

Your Tasks:

  • Create a function called printForecast that takes an array as input
  • The function should display temperatures in the format: "... TEMPºC in X days ..."
  • Each temperature should show how many days in the future it's forecasted
  • The output should be a single string with all forecasts

Test Data:

Test Data 1: [17, 21, 23]

Expected Output 1: "... 17ºC in 1 days ... 21ºC in 2 days ... 23ºC in 3 days ..."

Test Data 2: [12, 5, -5, 0, 4]

Problem-Solving Approach:

  • Understand the problem thoroughly
  • Break it down into sub-problems
  • Write clean and efficient code
  • Test with both provided data sets
